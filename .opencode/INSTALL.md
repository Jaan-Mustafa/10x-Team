# 10x-Team — OpenCode Installation

## Prerequisites

- [OpenCode](https://github.com/opencode-ai/opencode) installed and configured

## Installation

1. Clone or copy this plugin into your project directory
2. Run OpenCode from the project root — it will automatically detect the `.opencode/` directory
3. Skills from `skills/` will be available during your session

## Usage

The plugin's skills are injected on session start. See `skills/using-10x-team/SKILL.md` for a list of available skills.

# Privacy Policy

**Last updated:** April 11, 2026

## Overview

10x-Team is an open source Claude Code plugin that runs entirely on your local machine. We do not collect, store, or transmit any user data.

## What We Collect

Nothing.

## How the Plugin Works

- All skills are markdown files loaded locally by Claude Code
- Project state files (`.10x/`) are stored in your project directory on your machine
- No data is sent to any external server, API, or analytics service
- No telemetry, no tracking, no cookies

## Third-Party Services

This plugin does not connect to any third-party services. It has no network dependencies.

## Data Storage

The only files created are `.10x/` project state files in your local project directory. You control these files entirely — you can read, edit, delete, or gitignore them at any time.
